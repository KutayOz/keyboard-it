How it works

MacBook keyboard
+ trackpad
🔒
encrypted LAN
Windows PC
keyboard + mouse
Encrypted end to end. Traffic secured with the Noise protocol.
Full input. Keyboard, mouse movement, and scroll — all forwarded.
Always at hand. Menu‑bar / system‑tray indicator with login autostart.
Correct layouts. Turkish‑Q & AltGr handled via position‑based key forwarding.

Setup in 4 steps

keyboard‑it Setup Wizard
Follow the steps to pair your two machines.
  1. 1
    Install on both machines
    Run the .dmg on your Mac and the .msi on your Windows PC.
  2. 2
    Pair them
    Set the same pairing key on both, then enter the other PC’s IP address.
  3. 3
    Grant permissions (macOS)
    Enable Accessibility and Input Monitoring in System Settings.
  4. 4
    Double‑tap Fn to activate
    Your keyboard and trackpad now drive the Windows PC. Double‑tap again to come back.
